This project is live at the following URl:

http://shopping-cart.live/index.php

Flow:
1-By default user will land on index page where he can view items but can't add them to cart or like for this he has to login or register these options are provided.
2-After successfull login user can use all the functionalites of features.

Features Added:
Connected to DB through php. Php mysql is used in the whole project backend. The databse values were stored in mysql database.
Ajax was used in connceting JS to the database. It allowed us to perform some small tasks without reloading the page for e.g like and adding to cart features.
Application is secured from sql injection. Php sanitization was done to prevent the users from an sql injection.
PHP sesions were also used in this project to keep track of user logins etc.
